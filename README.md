This script can filter the months, days and hours from the yearly .epw files.
These filtered files can be used for generating wind roses for specific months/days/hours.
The filtered epw files contain less than 8760 hours. Hence, these epw weather files cannot be used in energy modelling softwares.
The main purpose of the filtered wind rose is to model wind tunnel of CFD large scale CFD projects (outdoor cooling / urban modelling).
